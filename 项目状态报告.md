## 🚀 快速启动指南

### 1. 环境准备
```bash
# 确保已安装Docker和Docker Compose
docker --version
docker-compose --version
```

### 2. 启动项目
```bash
# 复制环境变量文件
cp env.example .env

# 编辑.env文件，配置必要的API密钥
# 启动所有服务
docker-compose up -d
```

### 3. 访问服务
- **前端**: http://localhost:3000
- **后端API**: http://localhost:8000
- **API文档**: http://localhost:8000/docs
- **数据库**: localhost:5432
- **Redis**: localhost:6379
- **MinIO**: http://localhost:9000
